Privacy Policy – YouTube Text Titles

Single Purpose

This extension provides a distraction-free overlay on YouTube showing only video titles, helping users select videos thoughtfully without clickbait.

Permissions and Justifications

  • Host permission (*://www.youtube.com/*): Required to run the extension only on YouTube pages to display video titles in a clean overlay.
  • Remote code: No remote code is used. All JavaScript runs locally in the user’s browser.

Data Usage

This extension does not collect, transmit, or store any user data. All functionality runs locally in the browser. Users’ browsing activity and personal information remain private and are not accessed by the extension.

Compliance

The extension fully complies with Chrome Web Store Developer Program Policies.
